Paw Run is an endless runner video game in which the player controlls a cat running on the streets collecting fishbones along the way as it also trying to avoid the obstacles ahead. The fishbones collected can be used to recruit and use more kittens to run with.

The game includes a character selection system that allows the player to choose the kitten they want to run with and collect the fishbones. Every kitten has its own value of fishbowns that is needed to be unlocked in order for the player to be able to choose that specific character. There is also data persistence in the game through a Save/Load system in order to keep track of how many fishbones are available and which characters are unlocked.

Paw Run was made with Unity.

This game was made solely with free assets from various artist that can be found onlne.
